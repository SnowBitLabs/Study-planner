 Smart Study Planner

A Python-based mini project that generates a personalized study plan based on energy level, available time, and subject strengths and weaknesses.


 Overview

Smart Study Planner is designed to simulate a real-life decision system that helps students decide:
- What to study
- How intensely to study
- When to rest

It dynamically adjusts suggestions using logical conditions based on user input.

 Features

- Energy-based study planning (scale of 1–10)
- Time-based decision making (0–12 hours)
- Personalized suggestions using:
  - Favourite subject
  - Weak subject
  - Easy and difficult topics
- Multiple study modes:
  - Light Mode
  - Controlled Session
  - Balanced Session
  - Productive Mode
  - Deep Work Mode
  - Beast Mode
- User feedback system
- Use of conditional logic with if-elif-else, and, or


 Technologies Used

- Python
  - Input handling
  - Conditional statements
  - Logical operators
  - Nested decision systems



 How It Works

The program takes:
- Energy level
- Available study hours
- Subject preferences

It processes these inputs through a structured decision system and outputs a customized study plan.
